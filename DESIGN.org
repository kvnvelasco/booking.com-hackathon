#+TITLE: The Drunkery: Design and Implementation

* Frontend

  - ReactJS + Redux
  - Google Maps API
  - Material UI

* Backend

  - Perl 5!
  - Mojolicious Framework

* API
   :PROPERTIES:
   :host:     localhost
   :port:     3000
   :pretty:   json
   :END:

   #+BEGIN_SRC http
   POST /search_by_city
   Content-Type: application/json

   { "text": "new york" }

Content-Type: application/json
